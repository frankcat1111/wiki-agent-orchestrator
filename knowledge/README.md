# Wikipedia方針ナレッジベース

## 概要

このディレクトリには、Wikipedia記事作成時に参照すべき公式方針のドキュメントが格納されています。

## ファイル一覧

- `Wikipedia削除の方針 - Wikipedia.md` (70KB) - 削除基準の詳細
- `Wikipedia即時削除の方針.md` (54KB) - 即時削除の基準

## トークン最適化のためのガイドライン

**重要**: これらの大きなファイルは `.cursorignore` で除外されています。

### エージェントでの利用方法

エージェントがこれらの方針を参照する必要がある場合：

1. **Read tool** で直接該当ファイルを読み込む
2. 必要なセクションのみを抽出して使用
3. 全文をコンテキストに含めない

### 主要な方針サマリー

#### 削除の方針（要約版）

**特筆性の基準：**
- 信頼できる情報源で有意に取り上げられている
- 単なる言及ではなく、実質的な記述がある
- 一次資料のみでは不十分、二次資料が必要

**即時削除の対象：**
- 明らかな著作権侵害
- 宣伝・広告目的のページ
- 攻撃的な内容、名誉毀損
- 検証不可能な内容

**削除されやすい記事：**
- 企業：単なる企業紹介、実績の羅列
- 人物：自己言及、信頼できる出典なし
- 事件：速報性のみで百科事典的価値なし

### 推奨アプローチ

記事作成時は以下を確認：
1. 複数の独立した信頼できる情報源があるか
2. 中立的な観点で記述されているか
3. 検証可能な出典が明記されているか
4. 宣伝・広告的でないか
