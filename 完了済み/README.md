## アーカイブの利用方法

過去のプロジェクトを参照する必要がある場合：

1. **個別ファイルの参照**: Read toolで直接ファイルを読み込む
   ```
   例: Read outputs/VIALA自由が丘/final.txt
   ```

2. **全体の圧縮**: 長期保管が必要な場合
   ```bash
   tar -czf archive-$(date +%Y%m%d).tar.gz 完了済み/
   ```

3. **削除**: 不要な場合は削除してディスク容量を節約
   ```bash
   rm -rf 完了済み/
   ```

## 新規プロジェクト作成時

新しいプロジェクトは `outputs/` ディレクトリで作業してください。

完了後、必要に応じて：
- 重要な成果物のみこのディレクトリに移動
- または外部ストレージ/クラウドにバックアップ
- 不要な中間ファイルは削除
