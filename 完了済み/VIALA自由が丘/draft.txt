'''VIALA 自由が丘'''(ヴィアラ じゆうがおか)は、東京都世田谷区奥沢に所在する美容室。株式会社BEAUTECHにより運営されている<ref name="hotpepper">{{Cite web |url=https://beauty.hotpepper.jp/slnH000255127/ |title=VIALA 自由が丘(ヴィアラ ジユウガオカ) |website=ホットペッパービューティー |access-date=2025-12-07}}</ref>。

== 概要 ==
東急東横線・東急大井町線「自由が丘駅」より徒歩30秒の位置にあり、セット面15席を有する<ref name="hotpepper" />。ホットペッパービューティーによると、営業時間は平日・土日祝10:00〜20:00、定休日は毎週火曜日および第三水曜日となっている<ref name="hotpepper" />。

同店によると、主要サービスとしてカット、カラー、パーマ、縮毛矯正、トリートメントを提供しており、「小顔カット」を特徴的サービスとして掲げている<ref name="rakuten">{{Cite web |url=https://beauty.rakuten.co.jp/s3010010743/ |title=VIALA 自由が丘 |website=楽天ビューティ |access-date=2025-12-07}}</ref>。

== 沿革 ==
運営会社である株式会社BEAUTECHは2012年に設立された<ref name="hotpepperwork">{{Cite web |url=https://work.beauty.hotpepper.jp/WC00000869/JP0000015532/ |title=株式会社BEAUTECH VIALA 自由が丘 |website=ホットペッパービューティーワーク |access-date=2025-12-07}}</ref>。

== 店舗情報 ==
ホットペッパービューティーによると、以下の情報が掲載されている<ref name="hotpepper" />:

* 所在地: 東京都世田谷区奥沢5-26-12 XAREA自由が丘ビル2F
* アクセス: 東急東横線・東急大井町線「自由が丘駅」より徒歩30秒
* 座席数: セット面15席
* スタッフ数: スタイリスト6名、アシスタント6名

同社によると、駐車場は設置されておらず、近隣のパーキングを利用する必要があるとされている<ref name="hotpepper" />。
