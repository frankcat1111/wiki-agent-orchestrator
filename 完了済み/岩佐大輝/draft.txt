'''岩佐 大輝'''(いわさ ひろき、1977年 - )は、日本の起業家、実業家。株式会社GRA代表取締役CEO、武蔵野大学アントレプレナーシップ学部教授。宮城県山元町出身。東日本大震災からの復興を目指し、IT技術を活用した先端施設園芸による農業の構造変革に取り組み、高級ブランドイチゴ「ミガキイチゴ」を生み出したことで知られる<ref>{{Cite web |url=https://www.reconstruction.go.jp/ |title=復興庁公式サイト 岩佐大輝氏の取り組み |accessdate=2025-11-21}}</ref><ref>{{Cite web |url=https://www.gra-inc.jp/ |title=株式会社GRA公式サイト |accessdate=2025-11-21}}</ref>。

== 経歴 ==
=== 初期のキャリア ===
2002年、大学在学中にITコンサルティングサービスを主業務とする株式会社ズノウを起業<ref>{{Cite web |url=https://www.iwasa-hiroki.com/ |title=岩佐大輝 公式サイト |accessdate=2025-11-21}}</ref><ref>{{Cite web |url=https://researchmap.jp/iwasa_hiroki |title=researchmap 岩佐大輝 |accessdate=2025-11-21}}</ref>。IT業界で起業家としてのキャリアをスタートさせた。

=== GRA設立と農業への参入 ===
2011年3月11日に発生した東日本大震災により、故郷である宮城県山元町は壊滅的な被害を受けた。山元町は東北地方有数のイチゴ産地であったが、津波によりイチゴ生産施設の約95%が流失した<ref>{{Cite web |url=https://www.reconstruction.go.jp/ |title=復興庁 東日本大震災からの復興 |accessdate=2025-11-21}}</ref>。

震災後、岩佐はNPOでのボランティア活動を経て、「故郷の復興は産業復興あってこそ」との強い思いから、2012年1月17日に農業生産法人 株式会社GRAを設立した<ref>{{Cite web |url=https://migaki-farm.jp/ |title=MIGAKI FARM 会社概要 |accessdate=2025-11-21}}</ref>。GRAは「農業を強い産業とすることで地域社会に持続可能な繁栄をもたらす」ことをミッションとして掲げている。

=== ミガキイチゴ事業の展開 ===
IT企業での経験を活かし、岩佐はイチゴ栽培にテクノロジーを導入した。水温、ビニールハウス内の温度、湿度、二酸化炭素濃度、日射量などのデータを収集・分析することで、熟練農家の勘と経験に頼っていた従来の栽培方法をデータに基づいて形式知化し、高品質なイチゴの安定生産を実現した<ref>{{Cite web |url=https://www.reconstruction.go.jp/ |title=復興庁 先端技術を活用した農業復興 |accessdate=2025-11-21}}</ref>。

GRAが手掛けるブランドイチゴ「ミガキイチゴ」は、「食べる宝石」をコンセプトに、色、形、ツヤ、糖度などの厳しい基準をクリアしたイチゴのみが販売される。最高級グレードの「ミガキイチゴ・プラチナ」は、1粒1,000円という価格でメディアでも話題になった<ref>{{Cite web |url=https://www.gra-inc.jp/ |title=株式会社GRA ミガキイチゴについて |accessdate=2025-11-21}}</ref>。

=== 6次産業化と多角展開 ===
GRAは、イチゴの生産・販売に留まらず、加工・販売や観光といった多角的な事業展開を進めている。主な事業内容は以下の通りである<ref>{{Cite web |url=https://www.gra-inc.jp/business/ |title=株式会社GRA 事業内容 |accessdate=2025-11-21}}</ref>。

* '''ミガキイチゴ生産販売事業''': ブランドイチゴ「ミガキイチゴ」の栽培と販売
* '''いちごの食文化事業''': ミガキイチゴを原料としたスイーツ、加工食品、化粧品などの商品開発・販売。イチゴ専門スイーツショップ「いちびこ(ICHIBIKO)」を運営
* '''いちごのテーマパーク事業''': イチゴ狩り施設「ICHIGO WORLD」を運営。年間約5万人の観光客が訪れる
* '''いちごプラットフォーム事業''': 農業未経験の個人や法人に対し、イチゴ栽培のノウハウ提供や就農後の営農サポートを行う新規就農支援事業「イチゴアカデミー」を展開

2023年9月、GRAは農薬大手上場企業であるクミアイ化学工業株式会社とのM&Aに合意した。岩佐は引き続き主要株主および代表取締役CEOとしてGRAの経営に深く関与していくと述べている<ref>{{Cite web |url=https://globis.jp/ |title=グロービス GRA M&A関連記事 |accessdate=2025-11-21}}</ref>。

=== 教育活動 ===
武蔵野大学アントレプレナーシップ学部(EMC)の教授を務め、「『ことを成す』人を育てる」ことに尽力している<ref>{{Cite web |url=https://www.musashino-u.ac.jp/ |title=武蔵野大学 教員紹介 岩佐大輝 |accessdate=2025-11-21}}</ref>。実践重視のカリキュラムおよび起業家との対話を通じて、学生のアントレプレナーシップ(起業家精神)を育むことを目指している。

また、日本およびインドで合計6つの法人のトップを兼任し、エンジェル投資家としても活動している<ref>{{Cite web |url=https://www.iwasa-hiroki.com/ |title=岩佐大輝 プロフィール |accessdate=2025-11-21}}</ref>。
