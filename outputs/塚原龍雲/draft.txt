'''塚原 龍雲'''（つかはら りゅううん、[[2000年]] - ）は、[[日本]]の[[起業家]]、[[インド]][[仏教]]僧。株式会社KASASAGI代表取締役社長。本名は'''塚原 大'''（つかはら だい）<ref name="shinsho_plus_column">{{Cite web |url=https://shinsho-plus.shueisha.co.jp/column/tsukahararyuun/32552 |title=工芸から学んだ、これからの生き方・働き方 |publisher=集英社新書プラス |accessdate=2025-12-01}}</ref>。僧名は龍雲。

「日本文化に宿る美意識を世界に橋渡しする」をミッションに掲げ、伝統工芸品のEC事業や、伝統技術を建築・空間デザインに応用する事業を展開している<ref name="forbes_interview">{{Cite web |url=https://forbesjapan.com/articles/detail/73852 |title=伝統工芸を救う、 KASASAGI塚原龍雲が描く職人たちの未来 |publisher=Forbes JAPAN |accessdate=2025-12-01}}</ref>。

== 来歴 ==
2000年生まれ。高校卒業後、ITスタートアップでの成功を目指して[[アメリカ合衆国]]へ留学<ref name="shueisha_online">{{Cite web |url=https://shueisha.online/articles/-/252778 |title=日本の伝統工芸品、市場規模870億円は大きい？ |publisher=集英社オンライン |accessdate=2025-12-01}}</ref>。留学中に日本文化の訴求力の高さに気づき、帰国を決意する<ref name="forbes_2024">{{Cite web |url=https://forbesjapan.com/feat/culture-preneurs30-2024/ |title=CULTURE-PRENEURS 30 2024 |publisher=Forbes JAPAN |accessdate=2025-12-01}}</ref>。

2020年、19歳で帰国し、株式会社KASASAGIを設立。創業からの3年間はキャンピングカーで日本を9周し、全国1000カ所以上の伝統工芸職人の工房を訪問して回った<ref name="shueisha_online" />。

2021年、インド・[[ナーグプル]]にて、インド仏教最高指導者である[[佐々井秀嶺]]上人のもとで得度し、出家。僧名「龍雲」を授かる<ref name="shinsho_plus_column" />。

現在は、伝統工芸品のECサイト「KASASAGIDO」の運営に加え、伝統技術を建材に応用する「KOGEI MATERIAL」、伝統工芸を用いた空間デザインを行う一級建築士事務所「KASASAGI STUDIO」（2025年設立）などを展開している<ref name="prtimes_studio">{{Cite web |url=https://prtimes.jp/main/html/rd/p/000000006.000118334.html |title=伝統工芸の技術を空間デザインに応用する「KASASAGI STUDIO」設立 |publisher=PR TIMES |accessdate=2025-12-01}}</ref>。

2025年11月、初の著書『なぜ日本の手仕事が世界を変えるのか 経年美化の思想』を出版した<ref name="shinsho_plus_interview">{{Cite web |url=https://shinsho-plus.shueisha.co.jp/interview/mori_tukahara/32338 |title=経年美化とサステナブルな世界【前編】 |publisher=集英社新書プラス |accessdate=2025-12-01}}</ref>。

== 人物・思想 ==
* 「経年美化（Keinen Bika）」という概念を提唱し、時間の経過を劣化ではなく価値の増幅と捉える思想をビジネスや著書を通じて発信している<ref name="shinsho_plus_interview" />。
* 職人たちとの交流を通じて、自然のリズムや「いのち」とともに生きる智慧としての手仕事を再評価し、それを現代社会に実装することを目指している<ref name="forbes_interview" />。
* [[Forbes JAPAN]]による「CULTURE-PRENEURS（文化起業家）」の一人に選出されており、経済的合理性と文化的価値の両立を追求している<ref name="forbes_2024" />。

== 主な事業 ==
株式会社KASASAGIを通じて以下の事業を展開している。
* **KASASAGIDO** - 伝統工芸品のオンラインショップ。
* **KOGEI MATERIAL** - 伝統技術を建材に応用する事業。
* **KASASAGI STUDIO** - 伝統工芸を用いた空間デザイン・設計を行う一級建築士事務所。
* **KOGEI HUB** - 伝統技術のサンプルパネルを取り揃えた拠点。
