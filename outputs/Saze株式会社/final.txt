'''Saze株式会社'''(サゼ かぶしきがいしゃ)は、東京都千代田区に本社を置く情報システム構築企業。2016年4月設立<ref name="shachomeikan">{{Cite web |url=https://shachomeikan.jp/corporations/24314 |title=Saze株式会社 の企業情報 |website=社長名鑑 |access-date=2025-12-09}}</ref><ref name="mynavi">{{Cite web |url=https://tenshoku.mynavi.jp/company/434140/ |title=Saze株式会社の会社概要 |website=マイナビ転職 |access-date=2025-12-09}}</ref>。代表取締役社長は伊藤康浩<ref name="shachomeikan" /><ref name="interview">{{Cite web |url=https://shachomeikan.jp/industry_article/4103 |title=業界毎特集記事－Saze株式会社 代表取締役社長 伊藤 康浩 |website=社長名鑑 |access-date=2025-12-09}}</ref>。

== 概要 ==
同社発表によると、情報システム構築に関わる業務全般を事業内容としており、コンサルティング、システム開発、保守・改善などを手がける<ref name="mynavi" /><ref name="doda">{{Cite web |url=https://doda.jp/DodaFront/View/Company/j_id__10166737315/ |title=Saze株式会社の転職・企業概要 |website=doda |access-date=2025-12-09}}</ref>。本社所在地は東京都千代田区神田平河町1番地 第3東ビル8F<ref name="shachomeikan" /><ref name="mynavi" />。
